abstract class Cli::CommandBase

  • Cli::CommandBase
  • Reference
  • Object

Overview

The base of command classes.

Your application should not directly inherit this class. Instead, use Command or Supercommand.

Direct Known Subclasses

  • Cli::Command
  • Cli::Supercommand
